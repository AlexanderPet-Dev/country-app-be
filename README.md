# country-app-be
Spring Boot Country Application

# Build Project

- In the path /countries-api
- Execute command : ./mvnw clean install package
- In the path countries-api/target there should be countries-api-0.0.1-SNAPSHOT.jar

# Run Docker for PG DB

- In the path /countries-api/docker/db 
- Run terminal command to start : bash ./start.sh
- Run terminal command to stop : bash ./stop.sh

# Run java app

- in the path /countries-api/target
- Run terminal command : java -jar countries-api-0.0.1-SNAPSHOT.jar
